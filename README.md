  - turn PDF to Excel 
  - simple UI design

-  使用方法與注意事項
   -  詳細的程式碼和參考資料放在 **程式碼與資料** 中，請下載整個資料夾，執行主程式 **main.py** 即可
   -  **PDD不良分類程式** 是已經打包好的執行檔，可以直接下載整個資料夾並按照說明執行
   -  數據的input請選擇25P1 or 25P2資料夾裡的PDF檔，其中個別資料夾裡勿選 (但數據可能牽扯到公司機密，在此不提供)
